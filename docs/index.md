# OntoUML According to the Machines

An unofficial, LLM-generated guide to OntoUML stereotypes.

## Scope

This site documents OntoUML stereotypes available in the OntoUML VP plugin.

The documentation is organized into two main sections:

- [Class Stereotypes](stereotypes/classes/index.md)
- [Relation Stereotypes](stereotypes/relations/index.md)

## Documentation approach

All stereotype pages are intended to be generated, reviewed, or revised by large language models.

Each page follows a common structure:

- Description
- Stereotype Profile
- Examples
- References
- Generation and Review Log

!!! warning "LLM-generated documentation"
    This project contains documentation generated by large language models. The content should be reviewed critically and should not be treated as authoritative OntoUML documentation.
