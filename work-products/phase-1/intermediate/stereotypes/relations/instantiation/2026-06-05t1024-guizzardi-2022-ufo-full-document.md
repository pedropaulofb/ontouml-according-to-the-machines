# Instantiation

## Source-Specific Description Contribution

In this source, instantiation is the relation, symbolized as `::`, connecting individuals or types to types. It is central to the formalization because both types and their instances are first-order citizens in the domain of quantification.

The source defines types through possible instantiation and uses instantiation to support both first-order and second-order typing. In the concept-evolution analysis, the same typed structure supports higher-order types whose instances are themselves first-order types.
