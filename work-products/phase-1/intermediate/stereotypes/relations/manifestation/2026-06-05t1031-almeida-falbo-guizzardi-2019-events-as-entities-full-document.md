# Manifestation

## Source-Specific Description Contribution

The source situates manifestation in UFO-B as the relation by which events are manifestations of dispositions. It characterizes a sub-theory connecting endurants and events by describing how events manifest dispositions, and it states that dispositions can inhere in other endurants.

The source also uses manifestation to explain `<<participation>>`: if an endurant participates in an event, then the event is either a manifestation of a disposition of that endurant or is composed of such a manifestation.

The source further notes a duality between relators and events that are their manifestations. This supports a modeling distinction between mutable endurant counterparts and immutable events.
