# Creation

## Source-Specific Description Contribution

The source treats creation as an event-endurant relation: events can bring about situations in which endurants are brought into existence. In the proposed pattern, creation events create endurants and have their own temporal boundaries. The creation time of an endurant is derived from the termination time point of the creation event.

This relation is part of the source's broader account of events as changes affecting endurants. Creation is one possible change promoted by an event, alongside destruction, property change, and participation.
