# Characterization

## Source-Specific Description Contribution

In this source, characterization relates an endurant type to a moment type. The source states that a type is characterized by moment types that inhere in its instances.

The formal account requires that instances of the characterized endurant type bear instances of the characterizing moment type, and that instances of the moment type inhere in a unique instance of the characterized type. For quality types, the source emphasizes uniqueness of inherence in the characterized type.
