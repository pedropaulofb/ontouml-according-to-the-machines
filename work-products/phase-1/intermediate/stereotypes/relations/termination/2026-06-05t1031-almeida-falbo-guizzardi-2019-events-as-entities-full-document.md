# Termination

## Source-Specific Description Contribution

The source introduces `<<termination>>` for representing the termination of an endurant under historical semantics. Because events and their dependent endurants accumulate, termination is not modeled as removal from the universe of discourse.

A `<<termination>>` association relates an event type to a class stereotyped `<<phase>>`. The target phase is instantiated by the terminated endurant when it takes on a historical nature. In that phase, the endurant has immutable properties comparable to those of past events.

The source frames termination as a phase change produced by event modeling under historical semantics, not as a general deletion or disappearance mechanism.
