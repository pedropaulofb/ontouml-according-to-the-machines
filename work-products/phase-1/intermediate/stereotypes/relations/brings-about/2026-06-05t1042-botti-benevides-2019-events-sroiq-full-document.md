# Brings About

## Source-Specific Description Contribution

The source defines `bringsAbout` as a relation from `Event` to `Situation` (p. 16-17, axioms a69, a95, a108). Each event brings about exactly one situation in the FOL formalization (p. 10, Section 3.6, S4). If an event brings about a situation, that situation obtains at the time point that is the event's end point (p. 10, Section 3.6, S2').

`bringsAbout` is central to the source's treatment of events as world changes. It links an event to the situation resulting from that event, complementing `triggers`, which links the pre-event situation to the event (p. 10, Section 3.6, S1'-S4).

The source also uses `bringsAbout` in the definition of direct causation: an event directly causes another event when a situation brought about by the first event triggers the second event (p. 10, Section 3.6, S6; p. 21, axiom a176). In the manifestation subtheory, a manifested disposition is present in both the triggering situation and the brought-about situation (p. 8, Section 3.3, D8).
