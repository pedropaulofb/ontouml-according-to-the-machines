# Brings About

## Source-Specific Description Contribution

The source defines `brings-about` as a relation from an event to a situation. An event brings about a situation when the occurrence of that event results in the situation obtaining in the world at the event's end point.

In the formal account, every event has a unique maximal situation that it brings about. This situation embodies the effects of the event at the moment the event ends.

`brings-about` is also used to define direct causation between events. If one event brings about a situation and that same situation triggers another event, then the first event directly causes the second under the source's restricted causation account.
