# Triggers

## Source-Specific Description Contribution

The source situates `triggers` in UFO-B as part of the relation between situations, dispositions, and events. Dispositions are triggered by situations, and events arise as manifestations of dispositions.

The source also links triggering to causation. If an event brings about a situation that activates the dispositions manifested as another event, then the situation triggers the latter event and the former event causes it.

Triggering is also used in the source's account of historical dependence: historical dependence can hold when one event brings about a situation that triggers another event, or brings about a situation necessary for such triggering.
