# Triggers

## Source-Specific Description Contribution

The source defines `triggers` as a relation from `Situation` to `Event` (p. 16-17, axioms a68, a94, a119). Each event is triggered by exactly one situation in the FOL formalization (p. 10, Section 3.6, S3). If a situation triggers an event, that situation obtains at the time point that is the event's begin point (p. 10, Section 3.6, S1').

For atomic events, triggering is linked to dispositions: a situation triggers an atomic event exactly when there is a disposition activated by that situation and manifested by the event (p. 7, Section 3.3, D3). This relates `triggers` to `activates` and `manifestedBy` in the source theory.

The source also uses `triggers` in the account of direct causation: an event directly causes another event by means of a situation brought about by the first event and triggering the second event (p. 10, Section 3.6, S6; p. 21, axiom a176).
