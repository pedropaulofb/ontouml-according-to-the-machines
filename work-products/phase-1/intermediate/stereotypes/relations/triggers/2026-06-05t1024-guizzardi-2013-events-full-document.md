# Triggers

## Source-Specific Description Contribution

The source defines `triggers` as a relation from a situation to an event. A situation triggers an event when the event occurs because that situation obtains.

The relation is constrained temporally: a situation that triggers an event obtains at the event's begin point. The source also postulates that each event occurrence has a unique triggering situation.

The paper connects triggering to dispositions. A situation triggers an atomic event when that situation activates the disposition manifested by the event. Thus, `triggers` links the obtaining of a situation to the manifestation of a disposition through an event occurrence.

The source also uses `triggers` with `brings-about` to define direct causation: a brought-about situation that triggers another event mediates a direct causal relation between the two events.
