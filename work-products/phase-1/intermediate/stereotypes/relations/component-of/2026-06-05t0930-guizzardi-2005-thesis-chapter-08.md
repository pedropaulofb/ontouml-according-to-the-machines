# Component Of

## Source-Specific Description Contribution

According to section 8.3.2 and table 8-4, `componentOf` is a parthood relation between two functional complexes. The source identifies it as the most commonly used conceptual parthood relation and proposes using the standard UML aggregation/composition notation to represent shareable and non-shareable variants of this relation.

The source assigns `componentOf` the meta-properties non-reflexivity, anti-symmetry, non-transitivity, and weak supplementation. It constrains both association ends so that the connected classes represent universals whose instances are functional complexes. For sortal universals, this requires being stereotyped as `kind` or specializing a `kind`; for mixin universals, this excludes specialization paths that classify quantities or collectives.
