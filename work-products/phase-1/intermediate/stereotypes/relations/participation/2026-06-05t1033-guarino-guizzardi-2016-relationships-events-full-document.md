# Participation

## Source-Specific Description Contribution

- The source uses participation as a formal relation linking endurants to perdurants (p. 11, footnote 13).
- It also treats event participation as non-homogeneous: participants, parts, and qualities can have different degrees or kinds of involvement in an event (p. 10).
- For relationship-centered modeling, events clarify the role of relata because roles are treated as ways of participating in an event, not as roles of the relationship itself (p. 11).
- This contribution situates Participation as the link needed when a model must represent contextual involvement in an event beyond the core relationship (pp. 11-12).
