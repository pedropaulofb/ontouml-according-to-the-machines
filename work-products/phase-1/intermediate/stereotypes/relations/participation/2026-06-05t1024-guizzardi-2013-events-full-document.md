# Participation

## Source-Specific Description Contribution

The source introduces participation as an object-relative view of events. Besides decomposing a complex event mereologically into subevents, the source partitions a complex event by separating the portion that is existentially dependent on each participant.

A participation is defined as the portion of an event that depends exclusively on a single object. Participations are derived from event parthood and existential dependence rather than treated as primitive; the source defines `participationOf` through exclusive dependence between a participation and an object.

Participations may themselves be atomic or complex. The source emphasizes that the participation view is orthogonal to the mereological view: a participation of an individual object in an event can be mereologically simple or complex.

For structural conceptual modeling implications, the source states that events represented in structural models should be connected to their participants via existential dependence. This entails immutability constraints on association ends connected to participant types, according to the source's discussion.
