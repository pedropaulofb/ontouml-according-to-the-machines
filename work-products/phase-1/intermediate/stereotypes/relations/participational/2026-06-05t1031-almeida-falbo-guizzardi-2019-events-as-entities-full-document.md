# Participational

## Source-Specific Description Contribution

The source introduces `<<participational>>` as a stereotype for event part-whole relations that decompose an event into participations. This is treated as a constructed decomposition into portions, projected from the whole event by considering formal dependence on specific endurants.

A participational portion is maximal with respect to the property under consideration and disjoint from other portions produced by the same criterion. The source states that the maximum cardinality at the association end attached to the participant is always one, reflecting the UFO-B rule that participations depend exclusively on a single endurant.

The source also states that event part-whole relations are restricted to event classes: an event cannot be in a part-whole relation with an endurant. Event part-whole relations follow extensional mereology, and the weak supplementation principle must be enforced by requiring the sum of lower bounds of parts to be at least two.
