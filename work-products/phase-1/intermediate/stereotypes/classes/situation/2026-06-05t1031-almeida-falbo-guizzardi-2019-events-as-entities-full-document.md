# Situation

## Source-Specific Description Contribution

The source situates situations in the UFO-B theory of change. Events map the world from situations that activate dispositions to situations brought about by the occurrence of events.

In the source's account, situations can activate dispositions whose manifestations are events. If an event brings about a situation that activates dispositions manifested as another event, then the situation triggers the latter event and the former event causes it.

This source-specific contribution only characterizes `situation` through its role in event change, triggering, and causation. The source does not provide a complete independent stereotype profile for situations.
