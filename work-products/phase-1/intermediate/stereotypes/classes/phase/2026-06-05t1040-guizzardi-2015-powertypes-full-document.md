# Phase

## Source-Specific Description Contribution

The source characterizes phases as anti-rigid types. An instance may move into or out of a phase without ceasing to exist, and phase membership depends on changes in intrinsic properties.

This characterization is applied to powertype instances in the source's modeling desiderata. The paper argues that an ontologically adequate account of powertype instances must allow them to undergo qualitative change and to instantiate contingent classifications. The variable-embodiment interpretation supports this because the same endurant can have different manifestations or property values across worlds or times while retaining identity.

For Phase 1 purposes, the source supports describing phase as an intrinsic-condition-based, anti-rigid classifier. It also supports using phases to represent contingent states of powertype instances when those instances are interpreted as endurants rather than as abstract universals or extensional sums.
