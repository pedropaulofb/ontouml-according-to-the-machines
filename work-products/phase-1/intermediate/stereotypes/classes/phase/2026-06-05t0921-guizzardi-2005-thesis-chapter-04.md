# Phase

## Source-Specific Description Contribution

A class stereotyped as `phase` represents the phased-sortal category of phases. In the source, phases are anti-rigid sortals that classify possible stages in the history of a substance sortal.

The source characterizes phase specialization through phase-partitions. The phases that specialize a given substance sortal constitute a complete and disjoint partition of that sortal: at any world, each instance of the restricted sortal falls under exactly one phase in the partition.

The proposed profile therefore requires phases to be defined as part of a partition and requires the restricted superclass of a phase-partition to be abstract. Each phase must have a unique `kind` as a supertype. Since phases are anti-rigid, they cannot serve as supertypes of `kind` according to the source's rigidity constraint.
