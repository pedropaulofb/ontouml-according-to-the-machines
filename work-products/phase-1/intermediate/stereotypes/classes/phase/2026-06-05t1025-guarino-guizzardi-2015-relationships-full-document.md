# Phase

## Source-Specific Description Contribution

The source characterizes phase as a contingent type distinguished by modal meta-properties. Unlike a kind, a phase can apply to an individual only for part of that individual's existence, and the individual can move into or out of the phase without losing its identity.

Because the source treats relationships and relators as endurants capable of genuine change, it allows phase classifications to be applied to relator instances. A relator can therefore have phase-specific classifications that reflect contingent states during its lifetime.

The source situates phase in the family of OntoUML and OntoClean distinctions that support taxonomic modeling. Once relators are accepted as endurants, phase patterns used for object types can also be combined with the relator and material-relation pattern.

For Phase 1 purposes, the source supports phase primarily as a modal classification mechanism for endurants, including relators, rather than as a final profile-level specification.
