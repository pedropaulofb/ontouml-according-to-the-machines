# Kind

## Source-Specific Description Contribution

The source uses kind as the sort of type that supplies identity for endurants. A kind is instantiated necessarily by its instances and provides the principle by which an individual is individuated, identified, persists through change, and is judged identical or non-identical across situations.

In this account, the kind fixes which properties of an individual are essential and which are accidental. It also determines which qualitative changes an instance may undergo while remaining the same individual. Since identity judgments must be supported across possible situations, the identity principle must come from a type necessarily instantiated by its instances.
