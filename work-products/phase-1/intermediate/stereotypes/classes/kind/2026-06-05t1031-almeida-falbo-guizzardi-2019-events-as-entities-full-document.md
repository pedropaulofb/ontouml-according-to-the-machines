# Kind

## Source-Specific Description Contribution

The source recalls the OntoUML/UFO-A characterization of `<<kind>>` as a type that classifies its instances necessarily and provides a uniform principle of identity for them. This contribution is used as background for distinguishing event types from endurant types in the proposed event extension.

The source treats `<<kind>>` as an endurant-type stereotype. Consequently, the event extension requires disjointness between classes stereotyped `<<event>>` and classes stereotyped with endurant-type stereotypes such as `<<kind>>`. A class must not simultaneously specialize an event type and an endurant type.

This source-specific contribution is limited to the background characterization of kinds and their incompatibility with event classification in the proposed extension.
