# Kind

## Source-Specific Description Contribution

The source uses kind as a modal category for types that apply necessarily to their instances. An instance of a kind cannot cease to instantiate that kind without ceasing to exist, because the kind supplies identity conditions for the individual.

The paper extends the relevance of kind beyond independent object types to relationship or relator types. Since relators are treated as full-fledged endurants, each relationship instance is an instance of a kind that accounts for its identity through time.

For relators, the associated kind constrains the admissible extent of qualitative change. This supports the source's view that a relationship can change over time while remaining numerically the same relationship.

The source therefore supports using kind-based taxonomic organization not only for object types but also for types of dependent endurants such as relators.
