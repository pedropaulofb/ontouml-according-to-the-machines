# Kind

## Source-Specific Description Contribution

A class stereotyped as `kind` represents a substance sortal. In the source, a substance sortal supplies the principle of identity for its instances, and a `kind` is the profile construct used to represent this category in conceptual specifications.

The source imposes two identity-related commitments on `kind`. Every object in a conceptual specification using the profile must instantiate a `kind` either directly or indirectly. In addition, an object cannot instantiate more than one ultimate `kind`, since this would assign incompatible principles of identity to the same represented individual.

As a rigid sortal, `kind` is constrained in the proposed profile so that its supertype is not a `subkind`, `phase`, `role`, or `roleMixin`. This reflects the source's postulate that a rigid universal cannot specialize an anti-rigid universal.
