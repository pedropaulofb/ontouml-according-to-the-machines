# Kind

## Source-Specific Description Contribution

In this source, a kind is a fundamental sortal endurant type. It supplies the uniform principles by which its instances are individuated, identified, and persist through qualitative change. The source treats kinds as rigid types: an instance of a kind instantiates that kind necessarily in all possible situations in which the instance exists.

The source also states a strong sortality commitment: every endurant necessarily instantiates a kind, and at most one kind. Non-kind sortals are presented as specializations of kinds, so a kind anchors the identity principle inherited by its sortal specializations.
