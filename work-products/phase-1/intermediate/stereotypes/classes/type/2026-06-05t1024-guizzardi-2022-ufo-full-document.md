# Type

## Source-Specific Description Contribution

In this source, types are first-class entities in the domain of quantification. A type is implicitly characterized as an entity that is possibly instantiated, whereas individuals are necessarily not instantiated.

The source uses the instantiation relation to connect both individuals and types to types. It explicitly allows first-order and second-order types, thereby supporting higher-order typing in which some types have types as instances. The broader UFO taxonomy then differentiates endurant types, perdurant types, substantial types, moment types, and more specific type categories.
