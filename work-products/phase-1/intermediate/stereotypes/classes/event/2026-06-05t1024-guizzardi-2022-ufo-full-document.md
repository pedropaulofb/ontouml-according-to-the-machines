# Event

## Source-Specific Description Contribution

In this source, events are treated as perdurants. Perdurants unfold in time by accumulating temporal parts, and the source notes that perdurants are often referred to as events for accessibility.

Events are modally fragile in the source's account: they cannot genuinely change while preserving numerical identity. Apparent event change is explained either as variation among temporal parts or as change in an underlying endurant of which the event is a manifestation. The source also connects events to the manifestation of dispositions.
