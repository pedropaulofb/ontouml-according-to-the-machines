# Event

## Source-Specific Description Contribution

- The source restricts event away from the generic sense of perdurant or occurrent and treats events as emerging from scenes by a focusing process (pp. 9-10).
- An event is individuated by a spatiotemporal region and a focus, where the focus is a collection of focal individual qualities; distinct foci can determine distinct events in the same region (pp. 10-11).
- Events are described as manifestations of individual qualities and, for relational events, as manifestations of qualities constituting the focal relationship (p. 11).
- In modeling, events are needed to represent contextual participants, participant roles, and temporal constraints around a relationship, rather than only the core relationship itself (pp. 11-12).
