# Phase Mixin

## Source-Specific Description Contribution

The source characterizes Phase Mixin as an anti-rigid mixin universal whose instances classify substantials of different kinds through common intrinsic properties. Phase Mixin differs from Phase because it applies across different kinds rather than within one sortal identity line (Section 2.1, p. 3).

As a mixin universal, Phase Mixin may not specialize sortal universals. As an anti-rigid mixin, a second-order type specializing Phase Mixin may have any Mixin Universal as base type: Category, Phase Mixin, or Role Mixin (Section 5.2.2, p. 12; Section 5.4, p. 18).

The source presents second-order Phase Mixin patterns as partitioning a mixin universal according to mutable intrinsic properties. This positions Phase Mixin as the higher-order counterpart of phase-like classification when the classified entities may instantiate different kinds (Section 5.2.2, p. 12).
