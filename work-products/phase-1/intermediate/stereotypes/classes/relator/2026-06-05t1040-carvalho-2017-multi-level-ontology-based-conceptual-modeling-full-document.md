# Relator

## Source-Specific Description Contribution

The source defines Relator as a moment depending on a plurality of individuals. Relator Universals have relators as instances and are connected to substantial universals through mediation (Sections 2.1, pp. 2-3).

Relator Universal is disjoint from Quality Universal and Mode Universal and cannot specialize a substantial universal. Therefore a second-order type specializing Relator Universal must categorize a Relator Universal base type (Section 5.3, p. 14; Section 5.4, p. 18).

The source supports second-order relator classifications using either intrinsic properties of relators or properties of the connected relata as criteria. It also describes how relator taxonomies can be reflected in role taxonomies through mediation: a role type may be categorized according to the type of relator borne by its instances (Section 5.3.3, pp. 16-17).
