# Relator

## Source-Specific Description Contribution

The source recalls `<<relator>>` as an endurant-type stereotype for existentially dependent entities corresponding, at a high level, to reified relationships in conceptual modeling. It also uses relators to contrast endurant-based relationship modeling with event modeling.

Relators participate in current relationship modeling through `<<mediation>>` and can ground ordinary `<<role>>` instantiation. The source contrasts this with historical roles, which are grounded in event participation.

The source also discusses a duality between relators and events that are their manifestations. Relators, as endurants, may change contingent properties while maintaining identity; events, by contrast, are immutable. This difference is presented as a methodological consideration for deciding whether to model a phenomenon as an event or as an endurant counterpart.
