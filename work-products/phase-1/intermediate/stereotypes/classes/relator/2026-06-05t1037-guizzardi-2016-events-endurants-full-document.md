# Relator

## Source-Specific Description Contribution

The source treats relators as relational endurants. A relator can be the entity to which an object identifier refers when an ongoing relational phenomenon can change while maintaining identity. In contrast, the corresponding event or process is an accumulated manifestation of the relator's particularized properties in a given historical unfolding.

The source emphasizes social relators as central for representing and controlling social reality in information systems. Relators can have life-cycle phases, induce roles, and be connected to events that create, suspend, continue, or terminate their manifestations. This source-specific contribution supports modeling mutable ongoing relational phenomena as relators rather than as events.
