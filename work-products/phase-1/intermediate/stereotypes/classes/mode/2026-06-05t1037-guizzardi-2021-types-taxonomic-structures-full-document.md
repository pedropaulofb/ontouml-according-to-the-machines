# Mode

## Source-Specific Description Contribution

According to the supplied source, `«mode»` is an OntoUML 2 ultimate-sortal stereotype for classes whose instances are modes (Section 4.1, p. 9). The paper treats modes as moments, specifically as one branch of intrinsic moments distinguished from qualities in the UFO taxonomy of endurants (Section 3, p. 6). Because the 2021 theory generalizes the modal classification of endurant types beyond substantial types, mode types may participate in sortal and non-sortal taxonomic structures rather than being limited to a single flat moment-type category (Introduction; Section 3, pp. 2, 7). In OntoUML 2, `«mode»` identifies the mode nature of the instances and serves as their ultimate sortal.
