# Mode

## Source-Specific Description Contribution

This source treats `mode` as a subtype of intrinsic moment: modes are existentially dependent individual moments, contrasted with relators as relational moments and with qualities as another kind of intrinsic moment (Section 2, pp. 3-4; Section 3, p. 8). The formal theory partitions intrinsic moments into modes and qualities, and defines mode types as endurant types whose instances are modes (Section 3, p. 8).

For OntoUML profile interpretation, the source is transitional. It states that in the current version of OntoUML, «mode» carried information about the nature of instances of a moment type, because the language was silent about modal properties of moment types (Section 4, p. 10). In the proposed OntoUML 2.0 profile, «mode» is discontinued and replaced at the ultimate-sortal level by a distinct stereotype for mode kinds (Section 4, pp. 10-11). Therefore, this source contributes to a `mode` page by clarifying the ontological nature of modes and by recording that the proposed profile separates instance nature from modal properties for mode types.
