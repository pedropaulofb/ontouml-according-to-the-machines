# Mode

## Source-Specific Description Contribution

In this source, a mode is an intrinsic moment that is not treated as a quality projected into a value space. Modes can bear their own moments, including qualities, and the category includes dispositions as well as externally dependent entities.

The source uses modes to account for particularized intrinsic properties that cannot be projected into value spaces. Externally dependent modes are a special case: they inhere in one entity while depending on another independent entity, and they are used to capture relationally grounded properties.
