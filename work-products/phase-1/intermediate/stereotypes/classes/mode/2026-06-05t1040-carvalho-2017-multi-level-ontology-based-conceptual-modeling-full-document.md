# Mode

## Source-Specific Description Contribution

The source classifies Mode as an intrinsic moment not directly related to measure structures, in contrast to Quality. Mode Universal has modes as instances; modes reify intrinsic properties inhering in substantials (Sections 2.1 and 5.3.2, pp. 2-3, 16).

As for other moment universals, Mode Universal is disjoint from Quality Universal and Relator Universal and cannot have a substantial universal as base type. Therefore a second-order type specializing Mode Universal must categorize a Mode Universal base type (Section 5.3, p. 14; Section 5.4, p. 18).

The source supports second-order mode classifications where properties of mode individuals supply the classification criteria. It also allows those criteria to be reflected in a substantial taxonomy through characterization, so that a substantial type can be classified according to the type of mode it bears (Section 5.3.2, p. 16).
