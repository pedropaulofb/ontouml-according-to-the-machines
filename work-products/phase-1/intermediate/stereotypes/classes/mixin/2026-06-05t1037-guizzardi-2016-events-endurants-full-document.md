# Mixin

## Source-Specific Description Contribution

The source contrasts sortal types with dispersive types, also called mixins. A sortal classifies instances that share the same kind; a mixin may classify entities belonging to multiple kinds. This distinction is used to explain why some contingent relational classifications should not be modeled as roles but as role mixins.

For Phase 1 purposes, the source contributes the general characterization of mixin as a non-sortal or dispersive type category applicable to endurants, with role mixin treated as a contingent and relationally dependent specialization of that category.
