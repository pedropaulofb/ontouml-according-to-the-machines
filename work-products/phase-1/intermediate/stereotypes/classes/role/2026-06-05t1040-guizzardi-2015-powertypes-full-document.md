# Role

## Source-Specific Description Contribution

The source characterizes roles as anti-rigid types whose instances may move into or out of their extension due to changes in relational properties. Role membership is therefore contingent and depends on participation in a relevant relational context.

The source applies this account to powertype instances by arguing that such instances must be able to bear contingent classifications. Under the variable-embodiment interpretation, a powertype instance can play roles while participating in material relations, because it is treated as an endurant with modal and relational properties.

For Phase 1 purposes, the source supports describing role as a relationally conditioned anti-rigid classifier. It also supports the use of roles for powertype instances when the modeled entity's classification depends on externally grounded relational circumstances.
