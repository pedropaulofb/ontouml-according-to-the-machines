# Role

## Source-Specific Description Contribution

This source treats `role` as an anti-rigid sortal whose instantiation depends on relational conditions (Section 2, pp. 4-5). Roles capture relational properties shared by instances of a kind, contrasting with phases, which are anti-rigid but relationally independent (Section 2, pp. 4-5). The paper's central revision is that roles are meta-types of endurant types in general, not only of substantial types; the source explicitly motivates applying roles to relator, mode, and quality types as well (Introduction, pp. 2-3; Section 4, p. 11).

In the proposed profile, «role» represents an externally dependent anti-rigid sortal (Section 4, p. 11). The supported constraints are that each «role» specializes an ultimate kind stereotype from the proposed profile, that a class cannot specialize more than one kind, and that rigid or semi-rigid classifiers cannot specialize anti-rigid classifiers such as «role» (Table 1, p. 13).
