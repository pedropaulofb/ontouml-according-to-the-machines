# Role Mixin

## Source-Specific Description Contribution

The source characterizes Role Mixin as an anti-rigid mixin universal that classifies substantials of different kinds through common relational properties. It is externally dependent, like Role, but applies across different kinds rather than within one sortal identity line (Section 2.1, p. 3).

As a mixin universal, Role Mixin may not specialize sortals. As an anti-rigid mixin, a second-order type specializing Role Mixin may have any Mixin Universal as base type: Category, Phase Mixin, or Role Mixin (Section 5.2.3, p. 13; Section 5.4, p. 18).

The source uses second-order Role Mixin patterns for cases where role-like classifications apply to entities of different kinds. The cross-level relation is categorization rather than necessarily partitioning, because the classification criteria are relational properties borne in a relational context (Section 5.2.3, p. 13).
